overflow

Each box has the same content but different overflow setting:

visible — text spills out of the box because it doesn't fit. Watch it overlap neighbors.
hidden — text is clipped at the edge. No scrollbars.
auto — scrollbars appear ONLY when needed. The cleanest choice for content that might overflow.

float — wrap text around an image

Originally, float was the way websites did multi-column layouts. Today we have flex and grid. But float is still the natural choice when you want body text to wrap around an element — like a magazine.

The parent has display: flow-root, which contains the floated child without any clearfix hacks.

Bonus: horizontal scroll snap

A modern pattern using overflow-x and scroll-snap.
